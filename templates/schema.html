Integration Contract

All Nebula Forge pipeline tools accept ECS-lite events at these endpoints:

  • DriftWatch — POST /api/analyze · POST /api/validate
  • ClusterIQ — POST /api/cluster
  • AtomicLoop — emits ECS-lite events on POST /api/run (normalize=true)
  • HuntForge — reads expected artifact fields from ECS-lite schema

Required fields: event.id, event.created, event.source_type, log.source_type. All other fields are optional and omitted when empty.

Pipeline call pattern: POST /api/normalize/batch → save to disk or pass events array directly to downstream tool.
